# TEA-GRATUITO
Plataforma web desenvolvida para facilitar o acesso a instituições, serviços e atendimentos gratuitos voltados ao Transtorno do Espectro Autista (TEA) em São Paulo. Reúne informações em uma interface moderna, intuitiva e responsiva, auxiliando famílias, cuidadores e profissionais na busca por serviços de apoio.
