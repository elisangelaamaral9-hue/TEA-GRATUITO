## 📖 Sobre o projeto

O **TEA Gratuito** é uma plataforma web desenvolvida para facilitar o acesso a **instituições, serviços e atendimentos gratuitos** voltados às pessoas com Transtorno do Espectro Autista (TEA) no estado de São Paulo.

A proposta é reunir informações confiáveis em uma interface moderna, intuitiva e responsiva, permitindo que famílias, cuidadores e profissionais encontrem rapidamente locais de atendimento e apoio.

Além do impacto social, o projeto demonstra conhecimentos em desenvolvimento Front-end, organização de dados, experiência do usuário (UX) e responsividade.

---

## ✨ Funcionalidades

- 🔎 Busca por instituições e serviços
- 🏷️ Filtros rápidos por categoria
- 🏥 Cards informativos
- 📍 Localização das instituições
- 📝 Formulário para sugestão de novos serviços
- ❓ FAQ
- ⬆️ Botão "Voltar ao topo"
- 📱 Layout responsivo
- ♿ Interface pensada para melhor acessibilidade visual

## 🚀 Como executar

Clone o repositório:

```bash
git clone https://github.com/elisangelaamaral9-hue/TEA-GRATUITO.git
```

Entre na pasta:

```bash
cd TEA-GRATUITO
```

Abra o arquivo **index.html** em qualquer navegador.

Para uma melhor experiência durante o desenvolvimento, utilize a extensão **Live Server** do Visual Studio Code.
