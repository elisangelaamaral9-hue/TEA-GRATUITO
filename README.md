🧩 TEA Gratuito
Um espaço de cuidado, informação e inclusão.

O TEA Gratuito é uma plataforma web desenvolvida para facilitar o acesso a instituições, serviços e atendimentos gratuitos voltados às pessoas com Transtorno do Espectro Autista (TEA) no estado de São Paulo.
O projeto reúne informações em uma interface moderna, intuitiva e responsiva, permitindo que familiares, cuidadores e profissionais encontrem rapidamente locais de atendimento, apoio e orientação.

📖 Sobre o projeto

Encontrar serviços especializados nem sempre é uma tarefa simples. Muitas famílias enfrentam dificuldades para localizar atendimentos gratuitos e informações confiáveis.

Pensando nisso, o TEA Gratuito foi criado para centralizar essas informações em um único ambiente, tornando a busca mais rápida, organizada e acessível.

Além de seu propósito social, o projeto demonstra habilidades em desenvolvimento Front-end, organização de dados, experiência do usuário (UX) e criação de interfaces responsivas.

✨ Funcionalidades
🔍 Busca dinâmica por nome, instituição ou tipo de atendimento
🏷️ Filtros rápidos por categoria
🏥 Cards organizados com informações das instituições
📍 Mapa interativo com localização dos atendimentos
📖 Seção "Sobre" explicando a proposta do projeto
➕ Formulário para sugestão de novas instituições
💡 Sugestão rápida de serviços
❓ FAQ com respostas para dúvidas frequentes
⬆️ Botão "Voltar ao topo"
📱 Interface totalmente responsiva
🎨 Design pensado para conforto visual e acessibilidade
🎯 Objetivos
  - Facilitar o acesso a serviços gratuitos para pessoas com TEA.
  - Centralizar informações confiáveis em uma única plataforma.
  - Promover inclusão por meio da tecnologia.
  - Oferecer uma experiência simples, intuitiva e acolhedora.

🚀 Como executar

Clone o repositório:

git clone https://github.com/SEU-USUARIO/tea-gratuito.git

Entre na pasta do projeto:

cd tea-gratuito

Abra o arquivo index.html no navegador ou utilize a extensão Live Server no VS Code para uma melhor experiência durante o desenvolvimento.
